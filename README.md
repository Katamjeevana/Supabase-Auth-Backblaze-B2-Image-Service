# Supabase Auth + Backblaze B2 Image Service

A full-stack application for user authentication with Supabase and image storage with Backblaze B2.

## Features

- User authentication (signup/login) with Supabase
- Image upload to Backblaze B2 cloud storage
- RESTful API with JWT authentication
- Docker containerization
- AWS deployment ready

## Prerequisites

- Node.js 16+
- Docker
- Supabase account
- Backblaze B2 account
- AWS account (for deployment)

## Local Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables in `.env`
4. Start the server: `npm run dev`
